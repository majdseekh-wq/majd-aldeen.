# موقع مجد الدين — دليل سريع

## 1. رفع الموقع على GitHub Pages

1. روح لحسابك على GitHub، أنشئ Repository جديد (أو استخدم نفس الموجود).
2. ارفع هذي الملفات والمجلد بنفس المجلد الرئيسي (الجذر):
   - `index.html`
   - `style.css`
   - `tokens.css`
   - `app.js`
   - `products.json`
   - مجلد `images/` (فيه `logo.png` و `favicon.png`) — لازم يصعد كمجلد كامل بنفس الاسم
3. من Settings → Pages، فعّل GitHub Pages على فرع `main`.
4. خلال دقايق رح يصير عندك رابط شكله:
   `https://username.github.io/repo-name/`

**مهم:** لما ترفع على GitHub بطريقة "Upload files" بالسحب والإفلات، تأكد إنك تسحب مجلد `images` كامل (مش بس الصور لحالها) عشان يحافظ على نفس المسار `images/logo.png` ويلاقيه الموقع صحيح.

---

## 2. قبل ما تنشر — خطوتين مهمتين بملف app.js

افتح ملف `app.js` بأي محرر نص (حتى من GitHub نفسه بالـ "Edit" ✏️):

### أ) رقم الواتساب
بأول الملف، غيّر:
```js
const WHATSAPP_NUMBER = "966566909707";
```
لرقمك الحقيقي بصيغة دولية بدون + أو 00 بالأول (يعني لو رقمك 0512345678 يصير 966512345678).

### ب) الإيميل التلقائي (نسخة احتياطية لكل طلب)
1. روح لموقع **formspree.io** وسجل حساب مجاني بإيميلك.
2. أنشئ "New Form"، واختار الإيميل يلي بدك الطلبات توصله.
3. هياخذلك رابط شكله: `https://formspree.io/f/xxxxxxxx`
4. حط هاد الرابط بمكان:
```js
const FORMSPREE_ENDPOINT = "https://formspree.io/f/YOUR_FORM_ID";
```

**ملاحظة:** لو ما ضبطت هاد الجزء، الموقع رح يشتغل تمام وتوصلك الطلبات على واتساب بس بدون نسخة احتياطية بالإيميل.

## 3. كيف تضيف / تعدل / تحذف صنف (بدون أي برمجة)

كل المنتجات بملف **`products.json`** فقط. افتحه بمحرر نص بسيط.

### لتعديل سعر أو اسم صنف موجود:
دور على الصنف وغيّر القيمة بس (يلي بعد النقطتين)، **بدون ما تلمس اسم الحقل أو علامات `" "` أو `,`**.

مثال — تعديل سعر هريسة سادة:
```json
"سعر_نص_كيلو": 14,
"سعر_كيلو": 28,
```
لو حبيت ترفع السعر لـ 16 و 30، غيّرها لـ:
```json
"سعر_نص_كيلو": 16,
"سعر_كيلو": 30,
```

### لإضافة صنف جديد (بيتسعّر بالوزن — نص كيلو/كيلو):
انسخ هاد القالب وحطه داخل أي فئة (بين `[` و `]` تبع "المنتجات")، وما تنسى فاصلة `,` بعد القوس المتوفر بالصنف يلي قبله:
```json
{
  "اسم": "اسم الصنف الجديد",
  "وصف": "وصف قصير عنه",
  "نوع_التسعير": "وزن",
  "سعر_نص_كيلو": 20,
  "سعر_كيلو": 40,
  "صورة": "رابط الصورة هنا",
  "متوفر": true
}
```

### لإضافة صنف جديد (يتسعّر بالحبة):
```json
{
  "اسم": "اسم الصنف الجديد",
  "وصف": "وصف قصير عنه",
  "نوع_التسعير": "حبة",
  "سعر_الحبة": 6,
  "صورة": "رابط الصورة هنا",
  "متوفر": true
}
```

### لحذف صنف:
امسح كتلته بالكامل من `{` إلى `}` المقابلة له (واحذف الفاصلة الزايدة إذا صار عندك فاصلة وحدها قبل `]`).

### لإخفاء صنف مؤقتاً (بدون حذفه نهائي):
غيّر `"متوفر": true` إلى `"متوفر": false` — وبيختفي من الموقع تلقائياً لين ترجعه.

### لإضافة فئة جديدة (مثلاً "مناسبات" أو "عروض"):
انسخ هاد القالب وحطه قبل القوس الأخير `]` تبع "الفئات":
```json
{
  "id": "new-category-id",
  "اسم": "اسم الفئة بالعربي",
  "المنتجات": [
    { "اسم": "...", "وصف": "...", "نوع_التسعير": "وزن", "سعر_نص_كيلو": 0, "سعر_كيلو": 0, "صورة": "...", "متوفر": true }
  ]
}
```

## 4. بالنسبة للصور

حالياً الصور مأخوذة من الإنترنت (صور مؤقتة عامة) عشان الموقع ما يطلع فاضي. لما تجهز صورك الحقيقية:

- ارفع الصورة على أي مكان مجاني (مثلاً GitHub نفسه بمجلد `images/`، أو imgur.com)
- خذ رابط الصورة المباشر
- استبدل قيمة `"صورة"` بالصنف المطلوب بهاد الرابط الجديد

---

## 5. ملاحظة عن أمان GitHub Pages

كل تعديل بترفعه عبر GitHub بيتحدث بالموقع تلقائياً خلال دقيقة أو دقيقتين، بدون أي تدخل إضافي منك.

احتفظ بنسخة من هذي الملفات الخمسة على جهازك أو Google Drive كـ backup، حتى لو حصل أي طارئ تقدر ترفعها بسرعة بأي استضافة تانية.
